# UltraBet

## Build With

* ASP.NET CORE 5.0 MVC
* MSSQL Server
* EF Core
* SignalR
* HangFire
* NUnit
* Moq

## Background processes
**Hangfire** has one registered jobs
* Every 1 min to collect data from external api and populate the database
